### Row Level Security (RLS) Policies

```sql
-- Example: Expenses table
ALTER TABLE bills ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Company scoped bills" ON bills
  FOR ALL USING (
    company_id IN (
      SELECT company_id
      FROM user_companies
      WHERE user_id = auth.uid()
    )
  );

-- This ensures:
-- 1. Users can only see bills for their companies
-- 2. Auto-enforced at database level
-- 3. Cannot be bypassed via API
```
